## Project Scope

The Phishing Email Detection Model is a Machine Learning application that identifies whether an email is phishing or safe based on its textual content. The project uses Natural Language Processing (NLP) and Machine Learning techniques to classify emails and improve cybersecurity awareness.



## Objectives

- Detect phishing emails.
- Classify emails as Safe or Phishing.
- Extract email text features using TF-IDF.
- Display model accuracy and confusion matrix.
- Demonstrate Machine Learning in cybersecurity.



## Technologies Used

- Python
- Pandas
- Scikit-Learn
- TF-IDF Vectorizer
- Logistic Regression
- Google Colab



## Features

- Email Classification
- Text Feature Extraction
- Accuracy Evaluation
- Confusion Matrix Display
- User Email Testing



## Project Workflow

1. Create email dataset.
2. Convert email text into numerical features.
3. Train Logistic Regression model.
4. Calculate accuracy.
5. Display confusion matrix.
6. Predict user email category.



## Expected Outcome

The model successfully classifies emails as PHISHING or SAFE based on textual content.
